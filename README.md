# 🚀 Modern Portfolio Practice

A sleek, professional, and high-performance portfolio built with the latest web technologies. This project serves as a comprehensive practice for mastering **Next.js 15**, **Tailwind CSS v4**, and modern UI/UX principles.

## ✨ Features

- **🌓 Dynamic Theming:** Full support for Light and Dark modes using `next-themes`.
- **📱 Fully Responsive:** Crafted to look stunning on mobile, tablet, and desktop.
- **⚡ Performance First:** Optimized fonts with `next/font` and image handling.
- **🎨 Smooth Animations:** Engaging scroll animations powered by **AOS** (Animate On Scroll).
- **📨 Functional Contact Form:** Integrated with **Resend** for seamless email delivery.
- **🔗 Quick Communication:** Fixed WhatsApp floating button for instant contact.
- **🧭 Smooth Navigation:** Native CSS smooth scrolling for a premium feel.

---

## 🛠️ Tech Stack

- **Core:** [Next.js 15](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Components:** [Shadcn UI](https://ui.shadcn.com/)
- **Icons:** [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)
- **Animations:** [AOS](https://michalsnik.github.io/aos/) & [Framer Motion](https://www.framer.com/motion/)
- **Email Service:** [Resend](https://resend.com/)
