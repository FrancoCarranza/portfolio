# 🚀 Next.js Portfolio Practice

¡Bienvenido! Este es un proyecto de práctica desarrollado para profundizar en el ecosistema de **Next.js** y **Tailwind CSS**. El objetivo principal fue construir un portafolio moderno, dinámico y totalmente responsivo, aplicando las mejores prácticas de desarrollo.

> [!NOTE]
> **Aviso:** Este es un proyecto estrictamente de práctica/aprendizaje y no representa mi portafolio oficial.

## 🛠️ Tecnologías Utilizadas

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Lenguaje:** TypeScript
- **Estilos:** Tailwind CSS v4
- **Iconos:** Lucide React & React Icons
- **Animaciones:** AOS (Animate On Scroll) & React Type Animation
- **Componentes:** Shadcn UI & React Multi Carousel
- **Temas:** Next Themes (Dark/Light mode)

## ✨ Características Principales

- **Smooth Scrolling:** Navegación fluida entre secciones mediante CSS nativo.
- **Modo Oscuro/Claro:** Implementación completa de temas con `next-themes`.
- **Diseño Responsivo:** Adaptado para móviles, tablets y escritorio.
- **Animaciones de entrada:** Uso de AOS para una experiencia de usuario más atractiva al hacer scroll.
- **Formulario de Contacto:** Estructura lista para integración.
- **Optimización de Fuentes:** Uso de `next/font` para un rendimiento superior.
